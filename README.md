# 🚀 Panduan Menjalankan Notebook Secara Lokal

### 1️⃣ Clone Repository

```bash
git clone https://github.com/arikodwianto/DataScience_UTS.git
```

### 2️⃣ Masuk ke Folder Repository

```bash
cd DataScience_UTS
```

### 3️⃣ Jalankan Jupyter Notebook

```bash
jupyter notebook
```

### 4️⃣ Buka File Notebook

Contoh:

```text
pertemuan1.ipynb
```

### 5️⃣ Jalankan Semua Cell

```text
Kernel → Restart & Run All
```

Pastikan seluruh cell berjalan tanpa error dan seluruh output tampil dengan lengkap.

---

# ☁️ Panduan Menjalankan Notebook Melalui Google Colab

1. Buka Google Colab.
2. Pilih menu **File**.
3. Pilih **Upload Notebook**.
4. Upload file `.ipynb` dari repository.
5. Jalankan seluruh cell dari atas ke bawah.
6. Pastikan tidak ada error dan seluruh output tampil.

# 🎯 Ringkasan Materi Setiap Pertemuan

### Pertemuan 1 – Pengenalan Data Science

Mempelajari konsep dasar Data Science, Python dasar, variabel, tipe data, percabangan, perulangan, dan fungsi.

### Pertemuan 2 – Struktur Data Python, NumPy, dan Pandas

Mempelajari struktur data Python serta penggunaan NumPy dan Pandas untuk pengolahan data.

### Pertemuan 3 – Data Cleaning

Mempelajari missing values, data duplikat, normalisasi data, serta deteksi dan penanganan outlier.

### Pertemuan 4 – Statistika Dasar dan Analisis Data

Mempelajari mean, median, modus, standar deviasi, distribusi data, analisis univariat, bivariat, dan korelasi.

### Pertemuan 5 – Visualisasi Data

Mempelajari pembuatan bar chart, histogram, scatter plot, boxplot, dan dashboard visualisasi menggunakan Matplotlib dan Seaborn.

### Pertemuan 6 – Persiapan Data

Mempelajari handling missing values, encoding data kategorikal, train-test split, scaling, dan data leakage.

### Pertemuan 7 – Pengantar Machine Learning: Regresi Linear

Mempelajari preprocessing, pelatihan model Linear Regression, evaluasi model menggunakan MAE, RMSE, dan R² Score.

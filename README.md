# 🚀 Cara Menjalankan Notebook

## 1. Clone Repository

```bash
git clone https://github.com/arikodwianto/DataScience_UTS.git
```

Masuk ke folder:

```bash
cd DataScience_UTS
```

---

## 2. Jalankan Jupyter Notebook

```bash
jupyter notebook
```

atau

```bash
jupyter lab
```

---

## 3. Buka Notebook

Contoh:

```text
pertemuan7/pertemuan7.ipynb
```

Kemudian pilih:

```text
Kernel → Restart & Run All
```

---

# ☁ Menjalankan di Google Colab

1. Buka Google Colab
2. Pilih Upload Notebook
3. Upload file `.ipynb`
4. Pilih:

```text
Runtime → Run All
```

Seluruh kode dapat dijalankan dari awal hingga akhir tanpa error.

# 📝 Kesimpulan

Melalui praktikum Pengantar Data Science Pertemuan 1–7, telah dipelajari alur lengkap proses analisis data mulai dari pengolahan data mentah, eksplorasi data, visualisasi, persiapan data untuk machine learning, hingga pembangunan model Regresi Linear.

Hasil pembelajaran menunjukkan bahwa kualitas data sangat mempengaruhi performa model. Teknik preprocessing seperti encoding, scaling, dan train-test split menjadi tahapan penting sebelum proses pelatihan model dilakukan.

Regresi Linear memberikan pemahaman dasar mengenai hubungan antar variabel dan bagaimana Machine Learning digunakan untuk melakukan prediksi berdasarkan data historis. Pembelajaran ini menjadi fondasi untuk mempelajari algoritma Machine Learning yang lebih kompleks pada tahap berikutnya.
